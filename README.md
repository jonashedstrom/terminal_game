# RoRo Terminal

A first playable browser prototype for a RoRo ferry and terminal handling game.

Open `index.html` in a browser to play. The prototype is a static single page app, so it does not need installation, a server, or external packages.

## Prototype scope

- Export and import mode switching
- Vehicle scanning
- First Place of Rest zone placement
- Ship loading deck slots
- Import work orders
- Five minute round timer
- Score, handled vehicles, work order progress, and placement accuracy

## Domain notes used in the prototype

The first iteration models a compact version of a RoRo logistics chain: gate receipt, yard storage, dockside vehicle/equipment processing, stern-ramp loading, and dispatch. Work orders are based on common processing services such as PDI, accessory fitting, washing, paint/mechanical repair, damage survey, battery checks, and receipt/dispatch.

Visual direction: original isometric terminal art with a warm, readable management-game feel. It should evoke classic city-builder clarity and modern mobile friendliness without copying a specific game's assets or exact style.
